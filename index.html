Aiming for Gemini

Overview

Project Genesis is a vertical-scrolling shoot ’em up game in which the player controls a spacecraft to destroy waves of enemies, attempting to attain the highest score possible.

Project Genesis is built using C++ with the Allegro 5 game library.

Features Implemented

Simple game mechanics

5 buttons to control a spaceship. Shoot down incoming waves of enemies. It's that simple.

Responsive, lag-free gameplay

Laggy gameplay is the greatest killjoy a gamer can ever have. We don't provide that killjoy.
