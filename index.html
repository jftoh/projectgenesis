Overview

Project Genesis is a vertical-scrolling shoot ’em up game in which the player controls a spacecraft to destroy a giant ship, attempting to attain the fastest time possible.

Project Genesis is built using Java with the LibGDX game library.

Download

You can download the game HERE.

Some Instructions

  1. Make sure your computer has Java Runtime Environment (JRE) installed.
  2. Click on "Download ZIP."
  3. Extract files.
  4. Click on ProjectGenesis.jar.
  5. Run with Java Platform SE Binary.

Features Implemented

Simple game mechanics

5 buttons to control a spaceship. Shoot down an enemy. It's that simple.

Responsive, lag-free gameplay

Laggy gameplay is the greatest killjoy a gamer can ever have. We don't provide that killjoy.

Bullet Hell

It may be just one enemy ship, but whatever it shoots out from its cannons will keep you on your toes.


What COULD be Done

Cross-Platform

Bringing the game onto Android devices was one of our main highlights, and (probably) one of the veto factors to attain the Gemini level of achievement. However, we were not able to complete the user feature in time, which was disappointing.


*NEW* Proposed Level of Achievement

Our team proposes that we should be granted the Vostok (Beginner) level of achievement.


In terms of basic requirements, we have completed LiftOff and have clocked more than 130 hours on developing the game.


Rather than creating a web application, both of us decided that we wanted to make a game from scratch using a game library (LibGDX), writing code in language familiar to us (Java). We eventually did it!


In terms of difficulty, we feel that developing our game was as challenging, if not harder, as compared to creating a web application. The number of mathematical formulae and game logic scenarios that we have meddled with, especially the bullet patterns, was absurd.


We hope that you would be convinced by our argument and grant us the 2 Stars that we need!
